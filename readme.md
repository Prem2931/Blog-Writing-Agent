# 📝 Blog Writing Agent

An AI-powered **Blog Writing Agent** that automatically researches topics, generates structured blog posts, and enhances content using Large Language Models (LLMs).

This project demonstrates how autonomous agents can assist with **content creation, research, and structured blog generation**.

---

# 🚀 Features

* AI-powered blog generation
* Automated topic research
* Prompt engineering experiments
* Blog improvement using iterative prompting
* Image generation integration
* Research-based content generation
* Frontend + backend architecture

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Blog-Writing-Agent.git
cd Blog-Writing-Agent
```

Create virtual environment:

```bash
python -m venv myenv
```

Activate environment:

Windows:

```bash
myenv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file and add your API keys:

```
GROQ_API_KEY=your_groq_key
TAVILY_API_KEY=your_tavily_key
OPENAI_API_KEY=your_openai_key
```

---

# ▶️ Run the Application

Run backend:

```bash
python bwa_backend.py
```

Run frontend:

```bash
python bwa_frontend.py
```

Or run notebooks to explore each step of the agent workflow.
